# Adivina el Animal

Un juego interactivo donde la computadora intenta adivinar el animal en el que estás pensando.

## Cómo jugar

1. Abre `docs/Index.html` en tu navegador
2. Piensa en uno de los 5 animales disponibles
3. Presiona "Start" y responde las preguntas con Sí o No
4. Presiona "Resultado" para ver la respuesta

## Animales

- Serpiente
- Jirafa
- Ballena
- Pez Payaso
- Ornitorrinco

Ninguno. Solo necesitas un navegador web para jugar.
